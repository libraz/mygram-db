# MygramDB v1.4.0 Release Notes

**Release Date:** 2026-03-16
**Type:** Feature & Bug Fix Release
**Previous Version:** v1.3.9

---

## Overview

Version 1.4.0 is a significant release introducing Unix domain socket support, Prometheus cache metrics, a structured benchmark suite, and bitmap-based filter indexing. This release also includes extensive bug fixes across cache correctness, thread safety, binlog replication, and data integrity, along with a comprehensive Python-based end-to-end test suite.

**Highlights:**

- **Unix Domain Socket Support** - Local connections via Unix sockets for reduced latency and improved security
- **Prometheus Cache Metrics** - Full observability into cache behavior with Prometheus-compatible metrics endpoint
- **Structured Benchmark Suite** - Python-based benchmark framework with anomaly detection and saturation analysis
- **Bitmap-Based Filter Index** - New FilterIndex for faster filter evaluation with batch sort key lookups
- **Comprehensive E2E Test Suite** - Python-based end-to-end integration tests covering replication, DDL, cache, and resilience
- **15+ Critical Bug Fixes** - Thread safety, cache correctness, binlog reconnection, GTID handling

## New Features

### 1. Unix Domain Socket Support

**Type:** Feature

Add Unix domain socket support for local connections, enabling lower-latency communication for co-located clients without TCP overhead.

- Server listens on configurable Unix socket path
- CLI (`mygram-cli`) supports Unix socket connections
- Client library (`mygramclient`) supports Unix socket connections
- New error codes for Unix socket operations

## Migration Guide

No migration steps required. This is a backward-compatible release.

### Recommended Actions

- Configure Unix socket path in `config.yaml` if you want to use local socket connections
- Review new Prometheus cache metrics at `/metrics` endpoint
- Monitor FilterIndex memory usage in `/health/detail`
- If experiencing binlog replication stalls, verify stale connection recovery is working after upgrade
